step 1 config jupyter lab

generate the config file

1
2

($User)$ jupyter lab --generate-config
Writing default config to: /home/$User/.jupyter/jupyter_lab_config.py

generate the password file

1
2
3
4

($User)$ jupyter lab password
Enter password: 
Verify password: 
[JupyterPasswordApp] Wrote hashed password to /home/$User/.jupyter/jupyter_server_config.json

copy the password

1
2
3
4
5
6

($User)$ cat /home/$User/.jupyter/jupyter_server_config.json
{
  "ServerApp": {
    "password": "sha1:$User"
  }
}%

modify the following contents

1

($User)$ vi /home/$User/.jupyter/jupyter_lab_config.py

1
2
3
4
5

c.ServerApp.ip = 'localhost' --> c.ServerApp.ip = '0.0.0.0'
c.ServerApp.password = ' ' --> c.ServerApp.password ='sha1:xxx'
c.ServerApp.open_browser = True --> c.ServerApp.open_browser = False
c.ServerApp.port = 8888 --> c.ServerApp.port = xxxxx
c.ServerApp.notebook_dir = '' --> c.ServerApp.notebook_dir = '/home/$User/workspace/'

step 3 config service.conf

1
2

(root)$ cd /etc/supervisord.d
(root)$ nano service.conf

add the following content to service.conf/

 2
 3
 4
 5

[program:jupyter_xxx_gs1]
user = $User
command = zsh /etc/supervisord.d/xxx.sh
startsecs = 0
stopwaitsecs = 0
killasgroup=true
stopasgroup=true
autorstart = true
autorestart = true
redirect_stderr=true
stdout_logfile = /etc/supervisord.d/logfile.log
stderr_logfile = /etc/supervisord.d/xxx.err

step 4 config frpc

add jupyter lab domains

1
2

(root)$ cd /home/gs-1/frp_gs
(root)$ nano frpc.ini

add following content

1
2
3
4
5

[jupyter_xxx-gs]
type = http
local_ip = 127.0.0.1
local_port = xxxxx
custom_domains = xxxgs1.xxxlab.com

start frpc

1

(root)$ ./frpc reload -c frpc.ini

step 5 Tencent Cloud config

  1. Go to DNS 解析 DNSPod
  2. choose 解析
  3. choose 添加记录
  4. 主机记录 –> xxxgs1
  5. 记录值 –> xx.xx.xxx.xx
  6. choose 保存
